Supported Flow

QR login uses `/api/login/qr/key`, `/api/login/qr/create`, and `/api/login/qr/check`.

Phone login sends an access code through `/api/captcha/sent` and signs in through `/api/login/cellphone` with `phone`, `captcha`, and optional `countrycode`.

Status inquiry is available at `/api/logstatus` and `/api/login/status`.

The backend stores the upstream cookie in an `HttpOnly` same-origin session cookie.
